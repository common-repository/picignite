== Description ==

Pic Ignite is a unique way of monetizing your content, we turn images into money!

Its is a product that offers first and foremost the ability to automatically place geo-targeted, display adverting which overlays existing post images. With this level of focus we are able to bring users highly relevant ads based on the niche they are reading about.

**How Does It Work?**
Our system scans your page for relevancy to our available advertising. Then display banners are automatically linked to your images and turns them into ads. No linking or work required by you, our system does it for you!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `picignite` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the left-hand side menu, click on ‘Pic Ignite’
4. Register for Pic Ignite on the www.picignite.com website if you haven't done already.
5. Once your site is approved, add the CID, SID and HTML values you want to target for your ads.
